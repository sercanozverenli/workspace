Soru 6:
a = range(5)
x = 2
for a in a:
  x = x + x * a

Yukarıdaki kod bloğu çalıştırıldığında x’in değeri kaç olur?

A) 10
B) 50
C) 100
D) 240
E) 500
range(5) ifadesi, 0'dan 5'e kadar (5 hariç) olan sayıları içeren bir aralığı temsil eder. Yani, bu ifade bir dizi olarak [0, 1, 2, 3, 4] değerlerini üretir.

Başlangıçta x = 2 olarak tanımlanmıştır.
İlk döngü iterasyonunda, a değeri 0 olacaktır. İfadeden sonra x = x + x * a yerine x = 2 + 2 * 0 işlemi gerçekleştirilir ve x hala 2 olarak kalır.
İkinci döngü iterasyonunda, a değeri 1 olacaktır. İfadeden sonra x = x + x * a yerine x = 2 + 2 * 1 işlemi gerçekleştirilir ve x 4'e yükselir.
Üçüncü döngü iterasyonunda, a değeri 2 olacaktır. İfadeden sonra x = x + x * a yerine x = 4 + 4 * 2 işlemi gerçekleştirilir ve x 12'ye yükselir.
Dördüncü döngü iterasyonunda, a değeri 3 olacaktır. İfadeden sonra x = x + x * a yerine x = 12 + 12 * 3 işlemi gerçekleştirilir ve x 48'e yükselir.
Beşinci ve son döngü iterasyonunda, a değeri 4 olacaktır. İfadeden sonra x = x + x * a yerine x = 48 + 48 * 4 işlemi gerçekleştirilir ve x 240'a yükselir.
