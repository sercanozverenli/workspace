Soru 17:
using System;
class SiraliArama {
  public static int arama(int[] dizi, int x)
  {
    int N = dizi.Length;
    for (int i = 0; i < N; i++) {
      if (dizi[i] == x)
        return i;
    }
    return -1;
  }
  public static void Main()
  {
    int[] dizi= {
    85, 90, 46, 12, 15, 22, 35, 88
    };
    int x = 5;

    int sonuc = arama(dizi, x);
      Console.WriteLine(sonuc);
  }
}

Yukarıdaki kod bloğunun çıktısı aşağıdakilerden hangisidir?

A) 22
B) -1
C) 22
D) sonuc=5
E) sonuc
int[] dizi adında bir dizi oluşturulur ve bu dizi içinde belirli tam sayılar bulunur.

int x = 5; ifadesi ile x adında bir tam sayı değişkeni oluşturulur ve bu değişken 5 değerini alır.

arama adında bir fonksiyon tanımlanır. Bu fonksiyon, bir dizi ve bir hedef değeri (x) alır. Dizideki her elemanı sırayla kontrol eder ve hedef değeri bulursa bu elemanın dizideki konumunu döndürür. Eğer hedef değeri dizide bulunmazsa -1 değerini döndürür.

arama fonksiyonu dizi dizisini ve x değerini alır.

Bir döngü (for döngüsü) kullanılarak dizinin her elemanı tek tek kontrol edilir. Dizinin uzunluğu (N) kadar dönülür.

Her bir eleman dizi[i] ile kontrol edilir ve x ile karşılaştırılır. Eğer dizi[i] ile x aynıysa, bu durumda i değeri, hedef değerin dizideki pozisyonunu temsil eder ve bu i değeri arama fonksiyonu tarafından döndürülür.

Eğer dizi içinde x değeri bulunmazsa, döngü sona erer ve fonksiyon -1 değerini döndürür.

sonuc adında bir değişken tanımlanır ve arama fonksiyonu çağrılır. Bu durumda x değeri olan 5, dizide bulunmamaktadır, bu yüzden arama fonksiyonu -1 değerini döndürür.

Son olarak, Console.WriteLine(sonuc); ifadesi kullanılarak sonuc değeri ekrana yazdırılır. Bu nedenle çıktı olarak -1 alınır, çünkü 5 değeri dizide bulunmamaktadır.
