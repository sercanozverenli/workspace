{
int min, temp;
for (int dis = 0; dis <= enBuyuk; dis++)
{
min = dis;
for (int ic = dis + 1; ic <= enBuyuk; ic++)
if (arr[ic] < arr[min])
min = ic;
temp = arr[dis];
arr[dis] = arr[min];
arr[min] = temp;
}
}

Seçerek sıralama (Selection Sort) algoritması, n elemanlı bir diziyi sıralarken iç içe iki adet döngü kullanır: birincisi dizinin elemanlarını sırayla gezerken, ikincisi ise her bir elemandan sonraki elemanları karşılaştırır ve en küçük (veya en büyük) elemanı bulur. İşte bu nedenle bu algoritmanın zaman karmaşıklığı O(n^2) olarak ifade edilir
